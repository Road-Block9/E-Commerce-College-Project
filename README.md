# Mini E-Commerce Internship Project

A simple mini e-commerce project created for a college internship task. It uses product data from DummyJSON and includes a product listing page, product detail page, and cart page with bill summary.

## Tech Stack Used

- HTML
- CSS
- JavaScript

## APIs Used

- Product listing API: `https://dummyjson.com/products?limit=194`
- Product detail API: `https://dummyjson.com/products/{id}`

## Features

- Product listing page with clean product cards
- Product image, title, brand/category, price, discount, and rating
- Add to cart from listing page and detail page
- Increase, decrease, and remove cart items
- Clear cart option
- Bill summary with total items, subtotal, discount amount, delivery charge, and grand total
- Cart item count in navbar


## How to Run the Project Locally

1. Download or clone the project folder.
2. Open the project folder.
3. Open `index.html` in any modern web browser.
4. Browse products, view product details, and add items to the cart.

No server, backend, framework, or package installation is required.
